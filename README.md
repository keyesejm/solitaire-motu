# ⚔️ SOLITAIRE: Masters of the Universe ⚔️

A pixel-art inspired Klondike Solitaire game with epic MOTU vibes, smooth animations, and retro charm.

## Features

✨ **Three Epic Themes**
- Masters of the Universe (swords, fire, dragons, muscles, killer hair)
- Dark Fantasy (medieval dungeons, dragons)
- Neon Synthwave (retro neon aesthetic)

🎮 **Customizable Gameplay**
- Draw 1 or Draw 3 mode
- Volume control
- Undo/Redo with up to 20 moves
- Move counter

🎨 **Smooth Animations**
- Card flip animations
- Satisfying placement animations
- Theme transitions
- Victory celebration

🔊 **Sound Design**
- Procedural sound effects (flip, place, error, victory)
- Volume control

## Game Rules

Classic Klondike Solitaire:
- Build foundations from Ace to King by suit
- Tableau columns must alternate colors (red/black)
- Can place Kings on empty tableau spaces
- Drag and drop cards between tableau, foundations, and deck

## Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Vercel** - Deployment

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Build & Deploy

```bash
npm run build
vercel deploy
```
